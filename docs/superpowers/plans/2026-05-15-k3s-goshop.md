# GoShop on k3s (Oracle A1.Flex) Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Stand up a single-node k3s cluster on the existing Oracle A1.Flex VM, bootstrap the GitOps platform (ingress-nginx, cert-manager, ArgoCD, External Secrets), and deploy the `goshop` workload reachable at `https://goshop.cunghoclaptrinh.online`.

**Architecture:** k3s with traefik/servicelb disabled; ingress-nginx as a `hostNetwork` DaemonSet binding 80/443 on the host; cert-manager + Let's Encrypt HTTP-01 for TLS; ArgoCD App-of-Apps reconciling platform and workload from this repo; External Secrets Operator syncing from Doppler.

**Tech Stack:** k3s, Helm, ArgoCD, ingress-nginx, cert-manager, External Secrets Operator + Doppler, Bitnami Postgres & Redis charts, local-path-provisioner (bundled), Cloudflare DNS.

**Spec:** `docs/superpowers/specs/2026-05-15-k3s-goshop-design.md`

## Task 2: k3s install script

**Files:**
- Create: `clusters/k3s/install.sh`

- [x] **Step 1: Write installer**

```bash
#!/usr/bin/env bash
# Idempotent k3s bring-up for a fresh Oracle Ubuntu 22.04 A1.Flex VM.
# Run on the VM as a user with sudo.
set -euo pipefail

PUBLIC_IP="${PUBLIC_IP:?set PUBLIC_IP=<vm-public-ip> before running}"
DOMAIN="${DOMAIN:-goshop.cunghoclaptrinh.online}"

echo "==> Disabling swap"
sudo swapoff -a
sudo sed -i.bak '/ swap / s/^/#/' /etc/fstab

echo "==> Removing Oracle's iptables DROP rules (k3s needs them gone)"
sudo apt-get purge -y iptables-persistent netfilter-persistent || true
sudo iptables -F INPUT || true
sudo iptables -F FORWARD || true

echo "==> Installing k3s"
curl -sfL https://get.k3s.io | INSTALL_K3S_EXEC="\
  --disable traefik \
  --disable servicelb \
  --write-kubeconfig-mode 644 \
  --tls-san ${PUBLIC_IP} \
  --tls-san ${DOMAIN}" sh -

echo "==> Waiting for node Ready"
until sudo k3s kubectl get nodes 2>/dev/null | grep -q ' Ready '; do sleep 2; done
sudo k3s kubectl get nodes -o wide

echo "==> Done. Copy /etc/rancher/k3s/k3s.yaml to your workstation."
```

- [x] **Step 2: Make executable and commit**

